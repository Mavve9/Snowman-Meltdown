# Snowman-Meltdown ❄️

Entwickelt im Rahmen der Umschulung zum Softwareentwickler  

---

## Was ist das?

Ein klassisches Wort-Ratespiel als CLI-Anwendung in Python.  
Der Spieler muss ein verstecktes Wort Buchstabe für Buchstabe erraten —  
bevor der Schneemann schmilzt.

---

## Was wurde gelernt

- Aufbau einer interaktiven CLI-Anwendung
- Kontrollstrukturen, Schleifen und Bedingungen in Python
- Zustandsmanagement innerhalb eines Spielablaufs
- Versionskontrolle mit Git & GitHub

---

## Tech Stack

- 🐍 Python
- 🔧 Git / GitHub
